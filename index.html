Featured Project

Procter & Gamble — Three-Statement DCF Valuation

A fully integrated financial model built from scratch using P&G's public 10-K filings (FY2020–2025 actuals, FY2026–2030 forecasts). Income Statement, Balance Sheet, and Cash Flow Statement are cross-linked with automated reconciliation — if one cell changes, every downstream output updates and the balance check must return zero.

The model includes a complete DCF valuation using both Gordon Growth Model and Exit Multiple methodology, with a sensitivity analysis testing implied share price across 35 different assumption combinations.

Methodology

How I Built It

Data Collection

Extracted 6 years of historical financials from P&G's SEC 10-K filings. Every line item traced to the source document.

Three-Statement Model

Built linked IS, BS, and CFS with automated cross-references. Revenue drivers, working capital, CapEx, D&A, and debt schedules all flow through.

Assumption Engine

Dedicated assumptions sheet drives all forecasts. Revenue growth, margins, tax rates, and capital structure — change one input, see the full impact.

WACC Calculation

Cost of equity via CAPM (Rf: 4.5%, β: 0.6, ERP: 5.5%). Cost of debt from actual interest expense. Market-cap weighted capital structure.

DCF Valuation

5-year explicit forecast, terminal value via Gordon Growth (perpetuity) and Exit Multiple (16x EBITDA). Enterprise value bridge to implied share price.

Sensitivity Analysis

7×5 matrix testing WACC (5.5%–8.5%) against terminal growth rate (1.5%–3.5%). Colour-coded to show overvalued vs undervalued scenarios.

Key Insight

Where Does Enterprise Value Come From?

Explicit FCF Period (FY26–30) — 21%
$67,743M — The 5-year forecast period where we project each line item individually
Terminal Value — 79%
$261,102M — All value beyond Year 5, driven by one assumption: the terminal growth rate. Change it by 50bps and the implied price swings $15–20.

Why This Matters

Nearly 80% of P&G's enterprise value comes from a single assumption about long-term growth. This taught me that valuation isn't about getting THE answer — it's about understanding which assumptions drive the answer and testing them honestly. The sensitivity table below shows exactly how much the output changes when you adjust WACC and terminal growth rate simultaneously.
